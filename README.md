# Disinfectant_Marker
The repository is used to analyze multi-omics of pediatric and adolescent patients exposed to humidifier disinfectants, and to identify diagnosis biomarker for long-term lung injury.

## Description for folder
**A_**: Analysis of study using multi-omics data with clinical variables. 

**S_**: Additional analysis for Supplement Figures. 

**# Machine Learning analysis** was conducted using RapidMiner Studio, so its repository doesn't include this code.
